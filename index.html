Bienvenidas a mi Fan page sobre Sailor Moon

Etiqueta <h1>

La etiqueta <h1> se utiliza para definir el encabezado principal de una página web. Es el título más importante y usualmente se muestra con un tamaño de fuente grande y en negrita.

Etiqueta <h2>

La etiqueta <h2> se utiliza para definir un encabezado de segundo nivel. Es útil para subdividir secciones importantes dentro de una página web.

Etiqueta <p>

La etiqueta <p> se utiliza para definir un párrafo de texto en HTML. Es una forma de estructurar y organizar el contenido textual de una página web.

Etiqueta <img>

La etiqueta <img> se utiliza para insertar imágenes en una página web. Es importante incluir el atributo "src" con la ruta de la imagen y el atributo "alt" con una descripción de la imagen.

Etiqueta <ol>

La etiqueta <ol> se utiliza para crear listas ordenadas, donde cada ítem está numerado.

  1. Mercurio
  2. Venus
  3. Tierra
  4. Marte
  5. Júpiter

Etiqueta <ul>

La etiqueta <ul> se utiliza para crear listas desordenadas, donde los ítems no están numerados.

  • Usagi Tsukino (Sailor Moon)
  • Ami Mizuno (Sailor Mercury)
  • Rei Hino (Sailor Mars)
  • Makoto Kino (Sailor Jupiter)
  • Minako Aino (Sailor Venus)
